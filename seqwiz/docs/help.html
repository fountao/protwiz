SeqWiz: a modularized toolkit for next-generation protein sequence database management and analysis

  • Installation

    (1) . Required environments:

    Python 3.x, with pip and requests

    (2). Install Biopython for predicting physicochemical properties

    >>> pip install biopython

    *Recommended version: 1.7.x

    See: https://biopython.org/wiki/Download

    (3). Install wxPython for GUI supports

    Recommended version: 4.x
    For Windows:
    >>> pip install -U wxPython

    For Ubuntu 20.04:

    >>> pip install -U -f https://extras.wxpython.org/wxPython4/extras/linux/gtk3/ubuntu-20.04 wxPython

    #for libsdl supports, if needed

    >>> sudo apt-get install libsdl2-2.0
    >>> sudo apt-get install libsdl2-dev

    *For other Linux distributions, please find the correct versions from: https://extras.wxpython.org/wxPython4/extras/linux/

    See: https://www.wxpython.org/pages/downloads/

  • Functionalities

    App name Class Functions Note

    UpSpecies

    		 Core Search or view taxonomy ID Based on UniProt

    UpRetrieval

    		 Core Download species specific sequences and annotations, create structured database Based on UniProt

    UpConvert

    		 Core Convert FASTA from UniProt to PEFF or SQPD Based on UniProt

    CheckSeq

    		 Core Check the format of FASTA, PEFF, SQPD, SET or PEPLIS *PEPLIS: a list of peptide sequence

    DbManage

    		 Core Create structured database from non-UniProt sequences  

    SeqConvert

    		 Core Simple format converter for non-UniProt sequences  

    SeqFilter

    		 Core Sequence filter to generate a SET list from SQPD  

    SeqDecoy

    		 Core Generate decoy FASTA sequences  

    MatureSeq

    		 Core Generate mature forms of protein sequences Based on UniProt annotations

    SepFinder

    		 Core Predict sORF and SEPs from transcript sequences  

    SeqAnnotate

    		 Additional Sequence statistics for singular or grouped residues; calculation or prediciton of physicochemical properties, including: isoelectric point, physilogical charge, reduced molar extinction, cystines molar extinction, aromaticity, instability and grand average of hydropathy. *Require the biopython package

    TabFilter

    		Additional Table filter to generate a SET list *Recommend to use the result table from SeqAnnotate

    MotifCount

    		 Additional Motif count  

    SeqWindow

    		 Additional Sequence window extraction from a position table (for a list of sites or peptides)  

     

  • Usage

    (1). CMD usage

    Each tool of SeqWiz is designed with standard CLI interface, with self-describing arguments. Use the common "-h" flag to show the help in CLI:

    python {script_name}.py -h

    Reference usages:

    (1). UpSpecies

    #update species data

    >>> python UpSpecies.py -m update

    #search for Human taxonomy ID

    >>> python UpSpecies.py -m search -s Human -k E

    #view species data

    >>> python UpSpecies.py -m view

    (2). UpRetrieval

    #Browse UniProtKB column name for the usage of --addfts

    >>> python UpRetrieval.py -b

    #download FASTA sequences and features table from UniProt for HUMAN_9606, also generate SQPD and properties files (calls UpConvert.py and SeqAnnotate.py)

    >>> python UpRetrieval.py -t 9606 -s -a

    #download reference proteome from UniProt for MOUSE_10090

    >>> python UpRetrieval.py -t 10090 -r

    (3). UpConvert

    #create PEFF and SQPD files for download FASTA (using UpRetrieval.py)

    >>> python UpConvert.py {ipfi: input FASTA file} -o all

    (4).CheckSeq

    #check parsing rule

    >>> python CheckSeq.py {ipfi: input PEFF file} -t peff -r PEFF

    #check sqpd database

    python CheckSeq.py {ipfi: input sqpd file} -t sqpd -s

    (5). DbManage

    #create a database for contaminants based on a customized FASTA file, and convert to SQPD

    >>> python DbManage.py contaminants lab_2021 -m create -i {ipfi: input file} -f fasta -r Space -c -s

    #create a database for predicted SEPs based on a FASTA file (may be created by SepFinder), convert to SQPD and also generate annotation table

    >>> python DbManage.py mouse_SEPs lncRNA_2021 -m create -i {ipfi: input file} -f fasta -r Single -c -s -a

    (6). SeqConvert

    #convert FASTA file (as prepared by DbManage) to simple SQPD database

    >>> python SeqConvert.py {ipfi: input file} -t fasta2sqpd

    # extract FASTA sequences from SQPD based on a SET

    >>> python SeqConvert.py (ipfi: input file) -t set2fasta -o {odir: output dir}

    (7). SeqFilter

    #show avaliable feature names

    >>> python SeqFilter.py {dbfi: database file} -s

    #filter sequence by fragment annotation, note: there are "fragment" and "fragments" annotations from UniProt

    >>> python SeqFilter.py {dbfi: database file} -f Fragment -t pattern -e fragment

    #filter sequence by a motif of NX(X≠P)S/T/C/V (possible N-linked glycosylation sequence)

    >>> python SeqFilter.py {dbfi: database file} -f sequence -t pattern -e "N[^P][S|T|C|V]" -l nglyco

    (8). SeqDecoy

    #create decoy FASTA file for a SQPD

    >>> python SeqDecoy.py{dbfi: database file} -t sqpd

    (9). MatureSeq

    #generate mature sequences (also include the Full-length sequences) from a SQPD

    >>> python MatureSeq.py {dbfi: database file} -f

    (10). SepFinder

    #predict sORF and SEP sequences (using canonical and near-cognate codons, with ambiguous codon ) based on a fasta file of lncRNA trascript sequences, "Space" indicates that the header of the FASTA is space separated for extracting a unique ID

    >>> python SepFinder.py {ipfi: input sqpd file} -r Space -i N -a

    #predict sORF and SEP sequences (using only canonical codons) based on a fasta file of lncRNA trascript sequences

    >>> python SepFinder.py {ipfi: input sqpd file} -r Space -i C -a -l canonical

    (11). SeqAnnotate

    #sequence annotation of a SQPD database

    >>> python SeqAnnotate.py {ipfi: input file} -t sqpd

    #sequence annotation of a list of peptide (peplis should only contains peptides without a title row)

    >>> python SeqAnnotate.py {ipfi: input file} -t peplis -o {odir: output path}

    (12). TabFilter

    #select small protein entries in properties.tsv table as calculated by SeqAnnotate.py,

    >>> python TabFilter.py {tbfi: table file} -c 2 -e "[10,200]" -t numeric -n "[min,max]" -l small

    (13). MotifCount

    #counting the canonical motifs of cAMP-dependent protein kinase, using a list of input motif pattern

    >>> python MotifCount.py {dbfi: database file} -t sqpd -p "RR.S." "R.S." "KR..S."

    #counting motifs based on a list of motif

    >>> python MotifCount.py {dbfi: database file} -t sqpd -m (a file with a list of motif)

    (14). SeqWindow

    #extract sequence window for modified sites (START=END in position table), the position table file must contain a title row

    >>> python SeqWindow.py {dbfi: database file} {pofi: position table -l site

    #extract sequence window for peptide

    python SeqWindow.py {dbfi: database file} {pofi: position table file} -l pep

    (2). GUI usage

    SeqWiz also provides a GUI-to-CLI interface to run the tools.

    Step1: Start with the index window to show the list of tools.

    Step 2: Select the tool in the APP list and lanuch it

    Step 3: Set the parameters as needed

    Step 4: Click the "Assemble" button to check input and generate CMDs

    Step 5: Click the "Run" button to stat the mission

  • Practical examples

    (1). To create mature sequences for mouse proteins

    Step 1: change to the "core" directory:

    ./seqwiz/core/

    Step 2: find the taxonomy ID

    >>> python UpSpecies.py -m search -s Mouse -k E

    Step 3: download sequences and annotations

    >>> python UpRetrieval.py -t 10090 -s -a

    Step 4: generate mature sequences in FASTA format

    >>> python MatureSeq.py ../seqdbs/Uniprot/MOUSE_10090/2021_03/next/sqpd/SQPD.db -f

    Step 5: check the result file

    ./seqdbs/uniprot/MOUSE_10090/2021_03/classic/fasta/Single_mature.fasta

     

    (2). To predict SEPs derived from lncRNA transcripts (via GUI)

    Step 1: open the index window, choose the "SepFinder" in the APP list and launch the APP

    Step 2: choose the transcript FASTA file:

    Step 3: Choose the parsing rule

    Step 4: Choose the codon level

    Step 5: click "?" to show the help pop-up for initial codon level

    Step 6: click the "Assemble" to check input and generate CMDs

    Step 7: click the "Run" to stat the ruuning of this mission

    (3).  To generat subsets and retrieving sequences

    Step 1: change to the "core" directory:

    ./seqwiz/core/

    Step 2: generate a SET of reference proteome

    >>> python SeqFilter.py ../seqdbs/uniprot/MOUSE_10090/2021_03/next/sqpd/SQPD.db -f Proteomes -e " " -t exact -l proteome -m remove

    Step 3: generate a SET of small proteins, based on reference proteome

    >>> python ../adds/TabFilter.py ../seqdbs/uniprot/MOUSE_10090/2021_03/classic/table/properties.tsv -c 2 -l small -e “<200” -t numeric -n “<max” -i intersection  -r ../seqdbs/uniprot/MOUSE_10090/2021_03/next/sets/SQPD_filter_proteome.json

    Step 4: generate a SET of sequences with a speific motif, based on small SET

    >>> python SeqFilter.py ../seqdbs/uniprot/MOUSE_10090/2021_03/next/sqpd/SQPD.db -f sequence -e "N[^P][S|T|C|V]" -t pattern -i intersection -r -l motif ../seqdbs/uniprot/MOUSE_10090/2021_03/next/sets/properties_filter_small.json

    Step 5: convert all of the SETs into classic FASTA files

    >>> python SeqConvert.py ../seqdbs/uniprot/MOUSE_10090/2021_03/next/sets/SQPD_filter_proteome.json -t set2fasta -l proteome

    >>> python SeqConvert.py ../seqdbs/uniprot/MOUSE_10090/2021_03/next/sets/properties_filter_small.json -t set2fasta -l small

    >>> python SeqConvert.py ../seqdbs/uniprot/MOUSE_10090/2021_03/next/sets/SQPD_filter_motif.json -t set2fasta -l motif
